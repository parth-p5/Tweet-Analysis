# Streaming Tweets
Tweet Analyze wall made using Twitter streaming API and D3 Wordcloud

1. Clone the project 

2. <code>npm install</code>

3. open <code>config.js</code>

4. Place your access tokens and keys here

```
  twitter: {
    consumer_key: '',
    consumer_secret: '',
    access_token: '',
    access_token_secret: '',
    timeout_ms: 60*1000
  }
```

5. <code> node index.js </code>
